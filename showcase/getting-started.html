Getting Started

Install Sitara and build your first enterprise application in minutes.

Component Showcase

Installation

Three ways to add Sitara to your project. All options give you the full library — choose what fits your workflow.

html 
<!-- Place inside <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/sitara-ds/src/sitara.css">

The CDN build is a single pre-concatenated CSS file (~28 KB gzipped). No build step required.

bash 
npm install sitara-ds

Then reference the CSS in your project:

html 
<link rel="stylesheet" href="node_modules/sitara-ds/src/sitara.css">

Or import in a bundler (Vite, webpack, Parcel):

js 
import 'sitara-ds/src/sitara.css';

Download the repository, copy the src/ folder into your project, then link the main CSS file:

html 
<link rel="stylesheet" href="src/sitara.css">

Copying the source files lets you override tokens directly in src/tokens.css without writing additional CSS.

Your First Page

Minimal boilerplate for a themed page with the full app shell.

1

Add the theme attribute

Set data-theme on <html>. All design tokens update automatically. Omit the attribute for the default indigo theme.

html 
<html lang="en" data-theme="banking">  <!-- banking | fintech | insurance -->
2

Set up the app shell

The app shell wires together the banner, collapsible sidebar, and main content area. The toggle is pure CSS — no JavaScript.

html 
<!-- Hidden checkbox drives the CSS-only sidebar toggle -->
<input type="checkbox" id="st-sidebar-toggle">

<div class="st-app">

  <header class="st-banner" role="banner">
    <label for="st-sidebar-toggle" class="st-banner__hamburger">
      <span class="st-banner__hamburger-icon">
        <span></span><span></span><span></span>
      </span>
    </label>
    <a href="/" class="st-banner__logo">
      <span class="st-banner__logo-mark">AB</span>
      <span class="st-banner__logo-name">Acme Bank</span>
    </a>
    <div class="st-banner__spacer"></div>
    <div class="st-banner__actions">
      <!-- settings, notifications, user avatar -->
    </div>
  </header>

  <nav class="st-sidebar">
    <div class="st-sidebar__nav">
      <!-- navigation items -->
    </div>
  </nav>

  <main class="st-app__main">
    <div class="st-content">
      <!-- page content -->
    </div>
  </main>

</div>
3

Add sidebar navigation

Sidebar groups use native <details> for CSS-only collapsing. Each item has a label and optional tooltip for the collapsed icon-only state.

html 
<details class="st-sidebar__group" open>
  <summary class="st-sidebar__section-label">Accounts</summary>
  <a href="/dashboard" class="st-sidebar__item st-sidebar__item--active">
    <svg class="st-sidebar__item-icon" viewBox="0 0 24 24">...</svg>
    <span class="st-sidebar__item-label">Dashboard</span>
    <span class="st-sidebar__item-tooltip">Dashboard</span>
  </a>
</details>
4

Build your content area

Inside .st-content use the page header, grid, and component classes freely.

html 
<div class="st-page-header">
  <div>
    <h1 class="st-page-header__title">Accounts</h1>
    <p class="st-page-header__subtitle">Manage all customer accounts.</p>
  </div>
  <div class="st-page-header__actions">
    <button class="st-btn st-btn--primary">New Account</button>
  </div>
</div>

<!-- 12-column grid -->
<div class="st-grid st-grid--gap-4">
  <div class="st-col-8">
    <div class="st-card">...</div>
  </div>
  <div class="st-col-4">
    <div class="st-stat">...</div>
  </div>
</div>

Themes

Switch the complete colour system with a single attribute. Themes can be nested — apply a different theme to any sub-component.

Banking
data-theme="banking"

Deep navy + gold. Conservative, trustworthy — ideal for retail and commercial banking portals.

FinTech
data-theme="fintech"

Violet + cyan. Modern, bold — designed for digital wallets, trading platforms, and neobanks.

Insurance
data-theme="insurance"

Forest teal + amber. Stable, warm — suited for insurance portals, claims management, and agent dashboards.

Custom theme

Override tokens at :root (or inside a [data-theme="custom"] block) to create your own palette:

css 
/* my-theme.css */
[data-theme="my-bank"] {
  --st-color-accent:         #00529B;
  --st-color-accent-subtle:  #E6EFF7;
  --st-color-accent-text:    #003D73;
  --st-color-accent-on:      #FFFFFF;
  --st-color-accent-border:  #99C0DF;
  --st-font-sans:            'Barlow', sans-serif;
}
